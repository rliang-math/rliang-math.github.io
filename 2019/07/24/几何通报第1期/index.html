程牧炯:用Apollonius圆解罗老师几何一题

程牧炯:ZJGS新初三学生,善于解几何题.

题: 过$\triangle ABC$外心作$AB$垂线,交$AC$延长线于$P$,外接圆于$X$,$CB$,$HP$交于$E$.求证:(r是外接圆半径.)

这道题,程原本给出了两个解法,一个用圆幂与相似性来做,冗长,但直观.另外一个便是用Apollonius圆的解法.

解: 联结$CX$,$DX$,延长$PX$交$\odot X$于$X^{\prime}$,由对称性有以下几个结论:
结论1: $\angle 1=\angle 2.$
结论2: $XD=XC.$
注:在这里令$\angle APX = \angle 1,\angle HPB = \angle 2,\angle PAX = \angle 3,\angle XAC = \angle 4.$
由结论1,2,可推出结论3:
结论3: $\angle 1 = \angle 2,\angle 3 = \angle 4.$
从而$X$为$\triangle ABC$内心,同理$X$也为$\triangle BPD$内心,由内心性质$DX$平分$\angle PDE$,$CX$平分$\angle PCE$.从而设$=k,$并设$XE=a,XP=ka.$

由此可知$\odot O$上任一点与$X$的连线内平分PE.
注: 因为$DX,AX,CX,BX$皆内平分PE,可知$\odot O$为满足与$X$连线内分$PE$的点的轨迹.由熟知的定理知结论成立.$\square$
